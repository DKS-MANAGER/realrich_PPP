# realrich_PPP: Global Wealth Analysis

This repository analyzes the world's 50 richest people by adjusting nominal USD wealth for Purchasing Power Parity (PPP).

**Key Insight:** Results show massive "real" wealth gains for billionaires in emerging markets (India, China, Indonesia) when adjusted for local purchasing power, compared to their Nominal USD rankings.

##  Project Structure

- **Dashboard**: \wealth_dashboard/\ - Production-grade Streamlit application.
- **Analysis Notebook**: \ichest_listPPP.ipynb\ - Data processing and visual generation.
- **Data**: \wealth_dashboard/data/richest_ppp.csv\ - Processed dataset.
- **Outputs**: \data_output/\ and \charts/\ - Generated reports and visualizations.
